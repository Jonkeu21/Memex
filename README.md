# Memex

Self-hosted second brain on a Raspberry Pi: one Telegram bot ingests URLs, files, voice notes, and snippets; an autonomous worker files them into an Obsidian vault by taxonomy; a web dashboard and chat retrieve everything with cited sources from any device.

## Getting started

New to Memex? Start with the install guide. It walks from a freshly bought Pi 5 to a working bot, dashboard, and synced vault in 11 numbered phases.
